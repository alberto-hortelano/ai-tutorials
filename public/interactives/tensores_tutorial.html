Tensores: Tutorial Interactivo

La estructura de datos fundamental de PyTorch y Deep Learning

1. ¿Qué es un tensor?

Un tensor es una generalización de escalares, vectores y matrices a dimensiones arbitrarias. Usa el slider de rango para ver la progresión.

shape: ()
ndim: 0
numel: 1
# Escalar (rango 0) import torch x = torch.tensor(3.14) print(x.shape) # torch.Size([])

2. Shapes y dimensiones

La shape describe el tamaño de cada dimensión. Ajusta filas y columnas en el panel de controles.

shape: (3, 4)
numel: 12
x = torch.randn(3, 4) # matriz 3×4 con valores aleatorios print(x.shape) # torch.Size([3, 4]) print(x.numel()) # 12

4. Broadcasting

Cuando las shapes no coinciden, PyTorch expande automáticamente las dimensiones menores para que la operación funcione.

A: (3, 4)
+
b: (4,)
=
C: (3, 4)
A = torch.randn(3, 4) b = torch.tensor([1.0, 2.0, 3.0, 4.0]) C = A + b # b se replica en cada fila automáticamente print(C.shape) # torch.Size([3, 4])

5. Tensores en XCS236

Ejemplos concretos del curso

  • (1, 28, 28) — Imagen MNIST: 1 canal, 28x28 píxeles
  • (batch, C, H, W) — Batch de imágenes (ej: 64 imágenes RGB de 32x32)
  • (out_features, in_features) — Pesos de una capa nn.Linear
  • (batch, latent_dim) — Vector latente z en un VAE
  • La salida de un modelo generativo tiene la misma shape que los datos de entrada
# Batch de 64 imágenes MNIST x = torch.randn(64, 1, 28, 28) print(x.shape) # torch.Size([64, 1, 28, 28]) print(x.ndim) # 4 print(x.numel()) # 50176 # Flatten para pasar a capa linear x_flat = x.view(64, 784) print(x_flat.shape) # torch.Size([64, 784])

Operaciones clave que verás en las tareas

x.view(batch, -1) — Aplanar todas las dimensiones espaciales (flatten).
x.permute(0, 2, 3, 1) — Reordenar ejes, ej: (B,C,H,W) a (B,H,W,C).
x.unsqueeze(0) — Añadir una dimensión de tamaño 1 (útil para broadcasting).
x.squeeze() — Eliminar dimensiones de tamaño 1.
