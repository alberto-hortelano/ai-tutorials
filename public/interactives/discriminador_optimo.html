Discriminador Optimo D* y Logits h*

Ejercicio 3 — Que aprende el discriminador optimo de una GAN?

D*(x) = p_data(x) / (p_data(x) + p_theta(x))   |   h*(x) = log(p_data(x) / p_theta(x))

Parametros de p_theta

Distribuciones p_data y p_theta

p_data(x) = N(0, 1) fija
p_theta(x) = N(mu, sigma^2) movil

Discriminador optimo D*(x)

D*(x) = p_data / (p_data + p_theta)
D = 0.5 (equilibrio)

Logits optimos h*(x) = log(p_data / p_theta)

h*(x) = log-ratio de densidades
h = 0 (no puede distinguir)
